## 🧠 About Me

Sou desenvolvedor Full Stack com mais de 5 anos de experiência na criação de aplicações web escaláveis e de alta performance.

Tenho especialidade em backend com **PHP e Laravel**, projetando APIs robustas e sistemas bem estruturados utilizando padrões modernos como **microsserviços e multisserviços**.

Minha experiência inclui:
- Desenvolvimento de **APIs RESTful**
- Processamento assíncrono com **RabbitMQ**
- Otimização de **queries e performance de banco de dados**
- Implementação de **cache com Redis**
- Aplicação de **boas práticas (SOLID, Clean Code)**

Também possuo forte atuação em ambientes de produção utilizando:
- **Docker** para conteinerização
- **AWS (EC2, S3, Lightsail)** para infraestrutura escalável
- **CI/CD** com pipelines automatizadas para deploys seguros
